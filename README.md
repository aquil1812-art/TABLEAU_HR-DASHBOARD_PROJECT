# HR Dashboard & Human Resources Dataset Project

## Project Overview
This project demonstrates the creation of a **comprehensive HR Dashboard** in Tableau, powered by a **realistic HR dataset generated using Python**. The dashboard allows HR managers and executives to analyze workforce trends, employee demographics, salary patterns, and performance metrics.

The dataset contains **8,950 employee records** with attributes including employee ID, name, gender, hire date, department, job title, education level, salary, performance rating, birth date, overtime status, and termination date.  

---

## User Story
**As an HR manager, I want a comprehensive dashboard to analyze human resources data**, providing both summary views for high-level insights and detailed employee records for in-depth analysis.

---

## Dashboard Views

### **1. Summary View**
Provides high-level HR metrics:
- Total number of hired, active, and terminated employees.
- Trend visualization of hired and terminated employees over the years.
- Employee breakdown by department, job title, HQ vs branches, city, and state.

### **2. Demographics View**
Insights into workforce composition:
- Gender ratio visualization.
- Employee distribution across age groups and education levels.
- Correlation analysis between education level and performance rating.

### **3. Income Analysis View**
Focus on salary patterns:
- Salary comparison across education levels and gender.
- Age vs. salary analysis within each department to identify trends and discrepancies.

### **4. Employee Records View**
- Detailed list of all employees including name, department, position, gender, age, education, and salary.
- Users can filter based on any attribute for granular analysis.

---

## Python Dataset Generation
The HR dataset was generated programmatically using **Python**, **Faker**, **NumPy**, and **Pandas**. Features include:
- Realistic employee demographics and job-related attributes.
- Custom probability distributions for hire year, department, job title, and termination date.
- Salary calculation adjusted based on education level, gender, and age.
- Consistency checks to ensure logical alignment between birth date, hire date, and termination date.
- CSV output ready for Tableau visualization.
